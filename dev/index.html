Sysexits.jl

Sysexits.jl provides the system exit code constants as defined by <sysexits.h>.

Installation

To install this package, use the following command inside the Pkg REPL:

pkg> add Sysexits

After the package is installed, it can be loaded with:

using Sysexits

Example

The following example stops the program with the exit code defined by Sysexits.ExitCode. If all characters in str are within the ASCII range then the program stops with exit code 0 (Sysexits.ok), otherwise it stops with exit code 65 (Sysexits.dataerr).

import Sysexits

str = "abcdefgh"
try
    astr = ascii(str)
    println(astr)
    Sysexits.exit()
catch err
    showerror(stderr, err)
    println(stderr)
    Sysexits.exit(Sysexits.dataerr)
end
